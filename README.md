Este repositorio contiene el material complementario de la monografía 
"Diseño e implementación de un entorno de simulación de ataques y detección de incidentes (SIEM/XDR) basado en Wazuh para laboratorios de ciberseguridad".

El contenido incluye configuraciones del laboratorio, reglas utilizadas durante las pruebas, 
comandos empleados para reproducir los escenarios de validación y evidencias obtenidas durante la investigación.

Todos los escenarios fueron ejecutados en un entorno virtual aislado con fines exclusivamente académicos y de investigación.
